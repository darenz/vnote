# vnote:A terminal note book
#### 终端笔记管理脚本

### 安装：
`git clone git@github.com:darenz/vnote.git`

`cd vnote`

`source install.sh`

### 功能：
*  默认笔记路径：~/.mynotes
*  `nt`    打开最近修改的笔记
*  `nt -a` 添加笔记。默认文件名为当前时间，可加参数作为文件名
*  `nt -d` 删除笔记。列出笔记列表，选择序号删除
*  `nt -l` 列出笔记。选择序号打开
### 特点：
*  默认使用vim  
*  直接打开最近笔记
*  管理笔记只需两个命令
